# Lesson 3: Paths, Lines, & Joins

Path construction defines the geometry of vector outlines, but the rendering of those outlines depends on stroke expansion parameters. Master the math of joins, caps, and miter limits to render clean geometries.

---

## 1. Core Concepts

### Strokes vs. Fills
*   **Fill**: Paints the entire region bounded by a closed path.
*   **Stroke**: Paints a line of a specific width along the path. 

A stroke is not just a thin line. It is a full polygon generated by expanding the path geometry outward from its centerline along the normal vectors.

### Line Caps and Joins
The visual appearance of line endings and corners is governed by caps and joins:
*   **Line Caps**: Define the geometry of the start and end points of open paths.
*   **Line Joins**: Define how two connected segments meet at an angle.

---

## 2. Mathematical Foundations

### Stroke Centerline Expansion
When a path is stroked with a line width $w$, the rasterizer expands the geometry by $w/2$ perpendicular to the tangent vector on both sides of the path. If $\mathbf{p}(t)$ is the path curve and $\mathbf{n}(t)$ is the unit normal vector at $t$, the boundary curves of the stroke are defined by:

$$\mathbf{b}_{\pm}(t) = \mathbf{p}(t) \pm \frac{w}{2} \mathbf{n}(t)$$

### Line Caps
*   **Butt**: The stroke ends abruptly at the endpoint, perpendicular to the segment tangent.
*   **Round**: A semi-circle with radius $r = w/2$ is centered at the endpoint.
*   **Square**: The stroke extends beyond the endpoint by a distance of $w/2$ as a square cap.

### Line Joins & The Miter Limit
When two line segments join at an angle $\theta$, they can connect in three ways:
*   **Bevel**: The outer corner is sliced flat by a straight line connecting the outer stroke boundaries.
*   **Round**: The outer corner is rounded with a circular arc of radius $w/2$.
*   **Miter**: The outer stroke boundaries are extended until they intersect in a sharp point.

For sharp angles (small $\theta$), the distance from the path vertex to the miter tip ($d_{\text{miter}}$) grows extremely large:

$$d_{\text{miter}} = \frac{w}{2 \sin\left(\frac{\theta}{2}\right)}$$

To prevent this miter tip from extending to infinity when $\theta \to 0$, graphics engines enforce a **miter limit**. The ratio of the miter length to the line width is checked:

$$\frac{d_{\text{miter}}}{w} = \frac{1}{2 \sin\left(\frac{\theta}{2}\right)}$$

If this ratio exceeds the set `miterLimit` ($L$), the engine aborts the miter join and renders a **bevel** join instead:

$$\frac{1}{2 \sin\left(\frac{\theta}{2}\right)} > L \implies \text{Fallback to Bevel}$$

## 3. Code Demonstration

The following Swift code builds a zigzag path and draws it using three distinct configurations of caps, joins, and miter limits.

```swift
import Core
import Geometry
import Renderers

func drawStrokeConfigurationDemo() {
    var context = GraphicsContext()
    
    // Draw canvas outline
    context.setFillColor(Color(r: 0.95, g: 0.95, b: 0.95, a: 1.0))
    context.fill(Rect(x: 0, y: 0, width: 450, height: 180))
    context.setStrokeColor(Color(r: 0.1, g: 0.1, b: 0.1, a: 1.0))
    
    // Define path points for a V-shape
    let path = Path()
    var zigzag = path
    zigzag.move(to: Point(x: 40, y: 140))
    zigzag.addLine(to: Point(x: 90, y: 40))
    zigzag.addLine(to: Point(x: 140, y: 140))
    
    // Config 1: Butt Caps & Miter Joins
    context.saveGState()
    context.setLineWidth(16.0)
    context.setLineCap(.butt)
    context.setLineJoin(.miter)
    context.setMiterLimit(10.0)
    context.stroke(zigzag)
    context.restoreGState()
    
    // Config 2: Round Caps & Round Joins (shifted right by 150)
    context.saveGState()
    context.translate(by: 150.0, 0.0)
    context.setLineWidth(16.0)
    context.setLineCap(.round)
    context.setLineJoin(.round)
    context.stroke(zigzag)
    context.restoreGState()
    
    // Config 3: Square Caps & Bevel Joins (shifted right by 300)
    context.saveGState()
    context.translate(by: 300.0, 0.0)
    context.setLineWidth(16.0)
    context.setLineCap(.square)
    context.setLineJoin(.bevel)
    context.stroke(zigzag)
    context.restoreGState()
    
    // Render to output
    let renderer = SVGRenderer(width: 450, height: 180)
    do {
        let svg = try renderer.draw(context)
        print("Zigzag SVG:\n\(svg)")
    } catch {
        print("Error: \(error)")
    }
}
```
